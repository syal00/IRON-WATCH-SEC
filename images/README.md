# Images Folder

This folder contains all image assets for the Iron Watch Security website.

## Logo Files

- **mainlogo.png** - Main company logo (PNG format) - **CURRENTLY IN USE**
- **logo.svg** - Company logo (SVG format - backup/alternative)
- **logo.png** - Company logo (PNG format - optional backup)
- **logo.jpg** - Company logo (JPG format - optional backup)

## Usage

The website currently uses `images/mainlogo.png` as the main logo.

The logo is displayed in:
- **Header:** Small version (60px height) at top left
- **Hero Section:** Large version (200-320px width) in the center

## Updating the Logo

If you want to use a different logo file:

1. Place your logo file in this `images/` folder
2. Name it `mainlogo.png` (or update `index.html` to use a different filename)
3. Replace the existing `mainlogo.png` file
4. Hard refresh your browser (`Ctrl + Shift + R`) to see the update

## Logo Specifications

- **Recommended size for header:** 60px height (will scale proportionally)
- **Recommended size for hero section:** 200-320px height (will scale proportionally)
- **Format:** SVG (scalable), PNG (with transparency), or JPG
- **Color:** Gold (#d4af37) and Silver (#c0c0c0) on transparent or black background
